# 第八个月（宝宝会爬行）
不要吓唬，比如天黑，外面有坏人之类，影响胆子小，不敢独处。
可以摆弄积木
可以培养自主进食
多增加植物性和动物性蛋白质的补充，肉泥，下泥，肝泥。
不要纵容用哭的方式解决问题，不利于性格养成。
每日保证600-800的乳制品
吃完辅食后，记得喝水


# 第九个月（可以睡整觉，夜间不喝奶）
产生自我意识
继续联系爬行，不用急着走
不要吃膨化食品，爆米花，油条，腌制的食品，味精多的食品会影响智力发育，对肠胃负担。
不要用学步车，会造成O形腿
食物不可以太碎，不利于咀嚼，可以吃小饼干之类的，练习咀嚼。

# 第十个月（叫爸妈）
母乳仍是主食，辅食只是辅助
坚决不吃零食水果
吃东西要在桌椅上，不追喂
每天给宝宝讲绘本
多玩小手拍拍游戏

# 第十一个月（可以扶站）
给了玩具后，就不要打扰，培养专注力。
要尽量跟他多说话，用简短的语言和他交流。
# 第十二个月（可以走路拉）
鼓励宝宝多说话。
1岁的宝宝，辅食提供营养一半左右的量阶段
一些能磨牙的食品，面包片，苹果片，多锻炼。
