# 第四个月（练习翻身）
接种第三针灰疫苗和第二针百日破
此时不要一味的追求夜间少奶，如果总量达不到会影响生长发育的。
认识颜色，锻炼视力。
练习翻身
不要让宝宝在无人看管的情况下醒来，熟睡时不要让宠物靠近。
# 第五个月（长牙齿）
出牙期会吃手，此时不要急于添加辅食，因为肠胃系统要等6个月才发育完善。
会认人了，主动伸手找抱。
高兴时，会大笑。
不要看电子产品，保护视力。
# 第六月（添加辅食）
能听懂自己名字了
听音乐培养乐感。
辅食的循序的添加顺序，开始时高铁米粉，米糊半流实物粥，等7、9个月大添加菜末，根茎素菜-绿叶素菜-水果-肉蛋。注意不要基于添加山药，芒果，坚果，海鲜等容易过敏的食物
辅食注意，不要喝豆浆，不要用牛奶、酸奶代替配方奶。
一岁以内不要吃盐
练习靠着物品多坐立，进行腿部力量的训练，为日后爬行做准备。
# 第七个月（可以坐拉）
坐的比较稳，会在床上翻滚，开始学习爬行
此时对矿物质的需求 增加，逐渐添加辅食的种类和数量，果泥，菜泥，蛋黄泥。
